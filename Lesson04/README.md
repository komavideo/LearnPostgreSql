* PostgreSql的基础数据类型

## PostgreSql的基础数据类型

* 数值型：
  + integer(int)
  + real
  + serial
* 文字型：
  + char
  + varchar
  + text
* 布尔型：
  + boolean
* 日期型：
  + date
  + time
  + timestamp
* 特色类型：
  + Array
  + 网络地址型(inet)
  + JSON型
  + XML型
